Essencial

Claude Code

Anthropic

Lê o projeto, edita arquivos e roda comandos sozinho, direto no terminal ou no editor. Uso pra quase tudo hoje.

Claude Code Usage Tracker

Yahya Shareef

Mostra quanto eu já gastei de Claude Code direto na barra de status. Sem abrir outra aba pra descobrir a surpresa no fim do mês.

OpenAI

Python, R & Ciência de Dados

A base do trabalho com dados, em Python e em R: notebooks, formatação automática e o que evita rodar script na mão.

Essencial

Python

Microsoft

Suporte oficial a Python: roda, debuga e troca de ambiente virtual sem sair do editor.

Essencial

Pylance

Microsoft

IntelliSense rápido e checagem de tipos pra Python. Ajuda bastante quando o projeto cresce e fica difícil de navegar sem ele.

Black Formatter

Microsoft

Formata o código Python no padrão Black ao salvar. Acaba com a discussão sobre estilo de código.

Essencial

Jupyter

Microsoft

Roda notebooks .ipynb dentro do editor mesmo, com gráficos e variáveis aparecendo inline.

R

REditorSupport

Suporte à linguagem R: terminal integrado, ajuda inline e visualização de dataframe. Resquício da época do blogdown, mas ainda uso direto.

mechatroner

Colore cada coluna de um CSV/TSV com uma cor diferente e deixa rodar query tipo SQL direto no arquivo. Muda a vida de quem abre CSV cru o dia inteiro.

Atalhos de formatação, sumário automático e preview pra quem escreve em Markdown o dia inteiro.

markdownlint
